# Consumer Behavior Analysis & Sales Insights

## Project Overview

This project analyzes consumer shopping behavior to identify key factors influencing purchasing decisions, customer engagement, and repeat buying patterns.

The analysis helps businesses understand how discounts, reviews, seasons, and sales channels (online vs. offline) impact customer behavior and overall sales performance.

## Business Problem

A retail company is experiencing changes in customer purchasing patterns across demographics, product categories, and channels.

The goal of this project is to uncover actionable insights that can help improve customer satisfaction, increase sales, and build long-term customer loyalty.

## Objectives
Analyze customer purchase behavior
Identify key drivers of sales and repeat purchases
Segment customers based on buying patterns
Compare online vs offline performance
Provide data-driven business recommendations

## Tools & Technologies
Python (Pandas, NumPy) – Data Cleaning & Preparation
SQL (MySQL) – Data Analysis
Power BI – Data Visualization & Dashboard
Excel – Initial exploration
